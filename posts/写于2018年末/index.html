写于2018年末
Life
Nov 12, 2018
7min

这半年来经历了太多，原本9月从百度离职之后就打算写一篇心路历程，但是实在太忙了。后来打算11月10号最后一场托福考完之后再写，但是后来还是各种各样的事情，一直拖到了现在。我希望通过这篇文章记录一下这半年来的我跑过的各个地方，经历过的事，接触过的人。

本文起笔于2018年10月9日，结束于 2018年12月30日

本文最初我的计划是在11月10号考完托福之后，完成本文。但是各种事情拖来拖去的，加上考完试马上就是准备申请，一直没有机会写完本文。现在年末了，那么我就一点一点写完吧。

北京

从北京回来之后，我现在可以很明确的说，北京和北京之外的城市区别真的很大。虽然我很讨厌北京压抑的生活环境，但是北京有丰富的资源和成长空间。今年五月份春招的时候，基本也是怀着这样的想法去投简历的。春招期间我面试了猿题库，美团，百度，阿里，阿里是第一个面试的，当时真的年幼无知，投了淘宝前端组，还是第一个面试，不管是心里还是技术上都没有准备，不出所料一面（电话面）就挂了。

阿里挂掉之后就去面了猿题库，这个公司其实我觉得很有意思，早在我高考的年代，我就用过他们的产品，后来去了之后才了解到他们还有一个子公司叫粉笔。公司的文化也不错，业务是教育为主。其实去这里面试的时候我心理和技术都没有准备的很好，好在面试问题较为基础。当时问了我 Vue 的双向绑定原理，我也没有答上来，后来一些写程序的题，我做的也不是很好，很多是在帮助之下完成的。不过最后还是拿到了 offer，其实我自己评价起来的话，我觉得我能拿到 offer 并不是因为面试题，其实是因为从技术的广度比较大，并且有不错的潜力。我记得面试的时候跟面试官小姐姐分享了我当时正在发表的一篇论文，介绍了我是如何使用神经网络解决问题的。我记得当时那位小姐姐还非常开心，我估计她作为一个前端多半是不会神经网络那一套。其实这家公司很好，给的实习工资竟然能达到800/天，令人咋舌。

这场面试最后谈完了入职，当时给出的说法就是假期7月入职，面试完好像才4月的样子，还有很长时间。拿到 offer 之后人就容易飘，于是就打算去面试一些大公司，后来投了美团，也拿到了面试机会，同样也去了。第一次去美团觉得的很厉害，有好多楼，我去的比较早，到了之后围着美团的总部转了一圈。我清楚的记得当时为了缓解心里的压力，打趣的用非常乡土的山西口音说了句以后要是能给人家这单位打工可就好了。打趣归打趣，面试还是正常进行，来美团之前我一直在准备 JS 的知识，外加各种计算机基础知识，所以这场面试自身的知识水平明显提高了。经过了两轮面试，最后聊的还比较不错，最后回到了学校等通知。大概焦急的等了一周吧，接到电话发了 offer。说实话当时和我交流的美团的 HR 小姐姐真的不错，谈入职时间等事宜的时候非常顺利，我说7月入职，她把 offer 给我留到了六月底。

那么我确实得到了给这个单位打工的机会，之后还回了趟家，见了家人，还提到了这件事。那么人总归是想进步的，于是后来又去参加了百度的面试。说实话，当时拿到美团的 offer 之后，心里十分拒绝百度的面试，当时觉得美团就很好了。并且，百度之前的笔试题我做的很烂，百度和微软同一天同一时间笔试，我原本是没打算做百度的，可是打开微软的题发现不让用 IDE，不让查。当时我想用 Python 写几个简单的数据结构结果好多 API 都忘了，果断关掉了微软的笔试，去做百度的。我记得当时答的很一般，并且最后一道大题是手动实现一个 Promise，完全不会，交了白卷。说实话到现在我也没有好好研究清楚这个事情。但后来还是拿到了面试机会，这点其实是我没有预料到的。之前舍友也拿到了百度的面试，虽然跟我不是一个方向，不过还是得到了很多经验。我了解到早去早结束，所以面试当天很早就去了。

作为一名技术人员，当初无知，或多或少就是把公司的等级和它的面试的难易程度联系到了一起。这件事情后来入职百度之后，我也和我的同事交流过，身边的人也有类似的想法。

西二旗

2018年7月4日我入职了北京百度，开始了我的前端开发实习实习，也开始了我这半年来跌跌撞撞的探索。

优雅的Python环境搭建-Windows

Python
Nov 15, 2016

近期帮助女票的在她的 Windows 电脑上安装 Python，其实原本这就是一件特别简单的事情。但是介于她是个研究 Computer Science 的人，而且可能用诸如 Hadoop这样的神奇工具，我决定还是得用个 Linux 环境，那怎么办，装Linux？不，我们要优雅、稳定，Linux Friendly。

工具链列表

操作系统：Windows 10 工具链：VirtualBox，Vagrant，Ubuntu 16.04，Python 3，Cmder，zsh，oh-my-zsh，PyCharm

安装配置步骤

  1. 下载并安装工具链中所有软件
  2. 启动 PyCharm 配置远程调试，参考 JetBrain 的这个文章
  3. 文章沿着以下顺序进行： -> Using a remote interpreter -> Make sure that a remote interpreter is configured -> Configuring Remote Interpreters via Vagrant
  4. 新建项目，注意，项目目录一定要在 Vagrant 的文件夹下，这样才能和虚拟机内的文件系统交互。

效果

  • Python 远程调试稳定正常，支持断点

  • SSH 稳定，响应速度快，系统睡眠唤醒后 SSH Session 不会断开

  • 坑：在 PyCharm 内置环境中调试 Python 时，输入文字会再次显示一遍，目前还没有发现问题原因。
